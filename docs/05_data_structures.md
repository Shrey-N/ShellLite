# Data Structures

## Basic Types
- **Strings**: `"Hello"`, `'World'`
- **Numbers**: `1`, `3.14`
- **Booleans**: `yes`, `no` (or `true`, `false`)
- **Null**: `nothing`

## Lists
Collections of items.
```javascript
fruits = ["apple", "banana"]
say fruits[0] # apple
```

## Dictionaries
Key-value pairs.
```javascript
user = { name: "Alice", age: 30 }
say user["name"]
# or
say user.name
```

## List Operations
```javascript
# Natural syntax
list_of_nums = a list of 1, 2, 3
unique_set = a unique set of 1, 2, 2, 3
```
