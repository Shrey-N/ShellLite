# Modules

## Importing
You can organize code into multiple files.

```javascript
# Import generic file
use "utils.shl"

# Import system module
use "math"
say math.sqrt(16)
```

## Aliasing
```javascript
use "math" as m
say m.pi
```

## Standard Modules
- **math**: Standard math functions.
- **time**: Time tracking and delays.
- **random**: Random number generation.
